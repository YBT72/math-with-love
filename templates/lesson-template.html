🔍 הגדרה

מספר מרוכב הוא מספר שניתן לכתיבה בצורה:

z = a + bi
כאשר a, b מספרים ממשיים ו-i היא היחידה המדומה (i² = -1)
  • a נקרא החלק הממשי של z
  • b נקרא החלק המדומה של z
  • i היא היחידה המדומה כאשר i² = -1
🌟 דוגמאות
  • z₁ = 3 + 4i → החלק הממשי: 3, החלק המדומה: 4
  • z₂ = -2 + 7i → החלק הממשי: -2, החלק המדומה: 7
  • z₃ = 5 → מספר ממשי (החלק המדומה = 0)
  • z₄ = 2i → מספר מדומה טהור (החלק הממשי = 0)

➕ פעולות על מספרים מרוכבים

חיבור וחיסור:

(a + bi) ± (c + di) = (a ± c) + (b ± d)i
מחברים/מחסרים נפרד חלקים ממשיים וחלקים מדומים

כפל:

(a + bi)(c + di) = (ac - bd) + (ad + bc)i
זכרו: i² = -1
💡 דוגמה מפורטת - כפל

חשבו: (3 + 2i)(1 + 4i)

פתרון:

(3 + 2i)(1 + 4i) = 3·1 + 3·4i + 2i·1 + 2i·4i

= 3 + 12i + 2i + 8i²

= 3 + 14i + 8(-1)

= 3 + 14i - 8

תשובה: -5 + 14i

תרגיל 1: זיהוי חלקים

עבור המספר המרוכב z = -3 + 7i, מצאו:

  • א. החלק הממשי
  • ב. החלק המדומה

פתרון:

א. החלק הממשי: -3

ב. החלק המדומה: 7

תרגיל 2: חיבור וחיסור

חשבו:

  • א. (2 + 3i) + (4 - i)
  • ב. (5 + 2i) - (1 + 4i)

פתרון:

א. (2 + 3i) + (4 - i) = (2 + 4) + (3 - 1)i = 6 + 2i

ב. (5 + 2i) - (1 + 4i) = (5 - 1) + (2 - 4)i = 4 - 2i

תרגיל 3: כפל

חשבו: (1 + 2i)(3 - i)

פתרון:

(1 + 2i)(3 - i) = 1·3 + 1·(-i) + 2i·3 + 2i·(-i)

= 3 - i + 6i - 2i²

= 3 - i + 6i - 2(-1)

= 3 - i + 6i + 2

= 5 + 5i

  • א. z₁ + z₂
  • ב. z₁ - z₂
  • ג. z₁ · z₂

שאלה 2

חשבו את הביטויים הבאים:

  • א. (1 + i)²
  • ב. (2 - 3i)(2 + 3i)
  • ג. i³ + i⁴ + i⁵

שאלה 3

מצאו את כל הפתרונות הממשיים והמרוכבים של המשוואות:

  • א. x² + 4 = 0
  • ב. x² - 2x + 5 = 0
