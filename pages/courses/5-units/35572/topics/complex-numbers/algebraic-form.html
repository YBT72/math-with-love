📖 תיאוריה

הגדרת המספר המרוכב

מספר מרוכב הוא מספר מהצורה z = a + bi, כאשר:

  • a הוא החלק הממשי (Real part)
  • b הוא החלק המדומה (Imaginary part)
  • i הוא היחידה המדומה כאשר i² = -1
צורה כללית:
z = a + bi

סימונים ומושגים בסיסיים

עבור מספר מרוכב z = a + bi:

  • Re(z) = a (החלק הממשי)
  • Im(z) = b (החלק המדומה)
  • הצמוד של z הוא z̄ = a - bi

פעולות חיבור וחיסור

חיבור וחיסור מספרים מרוכבים מתבצע רכיב רכיב:

חיבור:
(a + bi) + (c + di) = (a + c) + (b + d)i
חיסור:
(a + bi) - (c + di) = (a - c) + (b - d)i

🔍 דוגמאות מפורטות

דוגמה 1: זיהוי חלקים

שאלה: עבור המספר המרוכב z = 3 - 5i, מצא את החלק הממשי, המדומה והצמוד.
פתרון:
1 זיהוי הצורה הכללית: z = a + bi כאשר a = 3 ו-b = -5
2 החלק הממשי: Re(z) = 3
3 החלק המדומה: Im(z) = -5
4 המספר הצמוד: z̄ = 3 - (-5)i = 3 + 5i
תשובה סופית: Re(z) = 3, Im(z) = -5, z̄ = 3 + 5i

דוגמה 2: חיבור מספרים מרוכבים

שאלה: חשב: (2 + 3i) + (4 - 7i)
פתרון:
1 זיהוי הרכיבים: z₁ = 2 + 3i, z₂ = 4 - 7i
2 חיבור החלקים הממשיים: 2 + 4 = 6
3 חיבור החלקים המדומים: 3i + (-7i) = -4i
תשובה סופית: 6 - 4i

דוגמה 3: חיסור מספרים מרוכבים

שאלה: חשב: (5 - 2i) - (1 + 4i)
פתרון:
1 זיהוי הרכיבים: z₁ = 5 - 2i, z₂ = 1 + 4i
2 חיסור החלקים הממשיים: 5 - 1 = 4
3 חיסור החלקים המדומים: -2i - 4i = -6i
תשובה סופית: 4 - 6i

🎨 ייצוג גרפי במישור המרוכב

המישור המרוכב מאפשר לנו לייצג מספרים מרוכבים כנקודות במישור, כאשר ציר X מייצג את החלק הממשי וציר Y את החלק המדומה.

💡 טיפים לזכירה

🔑
זכור: i² = -1, i³ = -i, i⁴ = 1 (המחזוריות חוזרת כל 4 חזקות)
⚠️
שגיאה נפוצה: החלק המדומה של 3 - 5i הוא -5 ולא -5i
✨
טריק: לבדיקה מהירה, המכפלה z · z̄ תמיד נותנת מספר ממשי חיובי

💪 תרגילים מונחים

תרגילים אלה יעזרו לך להתרגל על זיהוי רכיבי מספרים מרוכבים ופעולות בסיסיות. כל תרגיל כולל רמזים ופתרון מפורט.

הוראות:

  • נסה לפתור כל תרגיל בעצמך לפני הצגת הפתרון
  • השתמש בכפתור "רמז" אם אתה תקוע
  • הצג את הפתרון בהדרגה לבדיקה עצמית

תרגיל 1: זיהוי רכיבים

שאלה: עבור המספר המרוכב z = -4 + 7i, מצא את Re(z), Im(z) ו-z̄
רמז: השווה את z = -4 + 7i לצורה הכללית a + bi ואל תשכח שהצמוד מתקבל על ידי שינוי הסימן של החלק המדומה
1 השוואה לצורה הכללית: z = a + bi, כאשר a = -4 ו-b = 7
2 החלק הממשי: Re(z) = -4
3 החלק המדומה: Im(z) = 7
4 המספר הצמוד: z̄ = -4 - 7i
תשובה סופית: Re(z) = -4, Im(z) = 7, z̄ = -4 - 7i
טיפ לתרגיל זה: שים לב שהחלק המדומה הוא המקדם של i, ללא ה-i עצמו

תרגיל 2: חיבור מספרים מרוכבים

שאלה: חשב: (-3 + 2i) + (5 - 8i)
רמז: חבר בנפרד את החלקים הממשיים ואת החלקים המדומים
1 זיהוי: z₁ = -3 + 2i, z₂ = 5 - 8i
2 חיבור החלקים הממשיים: -3 + 5 = 2
3 חיבור החלקים המדומים: 2i + (-8i) = -6i
תשובה סופית: 2 - 6i
טיפ לתרגיל זה: שמור על הסדר: תמיד חלק ממשי + חלק מדומה

תרגיל 3: חיסור מספרים מרוכבים

שאלה: חשב: (6 - 3i) - (-2 + 5i)
רמז: שים לב לחיסור מספר שלילי! (-2 + 5i) פירושו לחסר -2 ולחסר +5i
1 פתיחת סוגריים: (6 - 3i) - (-2 + 5i) = 6 - 3i + 2 - 5i
2 חיסור החלקים הממשיים: 6 - (-2) = 6 + 2 = 8
3 חיסור החלקים המדומים: -3i - 5i = -8i
תשובה סופית: 8 - 8i
טיפ לתרגיל זה: כשמחסרים מספר שלילי, זה כמו לחבר את ההפוך שלו

🏠 תרגילי שיעורי בית

תרגילי הבית נועדו לחזק את ההבנה של המושגים שנלמדו ולהכין אתכם למבחן. פתרו בקצב שלכם ובדקו את התשובות.

הוראות להגשה:

  • פתרו את כל התרגילים במחברת בכתב יד ברור
  • הראו את כל שלבי הפתרון
  • בדקו את התשובות באמצעות הכפתורים למטה
  • מועד הגשה: השיעור הבא

תרגיל 3

7 נקודות
נתון z = 2 - 3i. חשב:
א) z + z̄
ב) z - z̄
ג) מה אתה יכול להסיק על סוג התוצאה בכל אחד מהסעיפים?
תשובה:
z̄ = 2 + 3i
א) z + z̄ = (2 - 3i) + (2 + 3i) = 4
ב) z - z̄ = (2 - 3i) - (2 + 3i) = -6i
ג) z + z̄ תמיד נותן מספר ממשי, z - z̄ תמיד נותן מספר מדומה טהור

תרגיל 4 (אתגר)

10 נקודות
מצא את כל הערכים של a ו-b (ממשיים) כך שיתקיים:
(a + 2i) + (3 - bi) = 7 + 5i
תשובה:
(a + 2i) + (3 - bi) = (a + 3) + (2 - b)i = 7 + 5i
השוואת החלקים: a + 3 = 7 ו- 2 - b = 5
לכן: a = 4 ו- b = -3
