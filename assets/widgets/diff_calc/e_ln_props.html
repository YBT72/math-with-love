🔺 פונקציות אקספוננציאליות

📉 פונקציות לוגריתמיות

אסימפטוטות:
• y = eˣ: y = 0 כאשר x → -∞ (משמאל)
• y = e⁻ˣ: y = 0 כאשר x → +∞ (מימין)
• y = ln(x): x = 0 כאשר x → 0+ (מימין לציר)
• y = ln(-x): x = 0 כאשר x → 0- (משמאל לציר)

🔄 סימטריה אוטומטית:
• eˣ + ln(x): ציר הסימטריה y = x
• e⁻ˣ + ln(-x): ציר הסימטריה y = -x
• כאשר נבחרות כל הפונקציות - שני צירי הסימטריה יוצגו יחד
